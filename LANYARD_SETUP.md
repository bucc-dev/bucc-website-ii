# 3d lanyard setup instructions

## missing asset: card.glb

the `card.glb` file is required for the 3d lanyard component to work properly.

### download the card.glb file

1. visit the react bits repository: https://github.com/birobirobiro/react-bits
2. navigate to: `src/assets/lanyard/card.glb`
3. download the `card.glb` file
4. place it in: `public/lanyard/card.glb`

**direct link:**
https://github.com/birobirobiro/react-bits/blob/main/src/assets/lanyard/card.glb

### current setup

- `public/lanyard/lanyard.png` - present
- `public/lanyard/card.glb` - missing (download required)

### once downloaded

the lanyard component will automatically work on the about page at `/about`

### optional: customize the card

you can edit the card.glb texture using this online editor:
https://modelviewer.dev/editor/
